### Overview
NeighborShare is a hyper-local mobile marketplace designed to foster community trust and resource efficiency in urban environments like Addis Ababa.

### Technical Highlights
* **Cross-Platform Performance:** Built with Flutter, utilizing advanced UI techniques like Sliver delegates for a native-feel experience.
* **Real-Time Infrastructure:** Integrated Firebase Firestore for instant data synchronization.
* **Secure Authentication:** Robust user system using Firebase Auth with SHA-256 fingerprint security.
* **Dynamic UX Logic:** Context-aware interface differentiating between "Owner" and "Requester" views.

### Challenges Overcome
* **Production Deployment:** Navigated complexities of signed Release APKs, ProGuard, and keystore management.
* **Data Integrity:** Engineered a robust image upload flow that handles network latency.
