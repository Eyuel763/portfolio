### Overview
Reunite Hub is a backend service built in **Django + DRF**, providing secure user onboarding and authentication via **JWT**. It includes a custom user model (email-based login + Ethiopian phone validation) and structured API endpoints to support future workflows like volunteer and moderator roles.

### Technical Highlights
* **Team Leadership:** Led a **9-person team** at the **Google Developer Group (GDG)** to deliver a full-stack humanitarian web app.
* **Authentication Architecture:** Built a Django REST backend with **JWT authentication** and refresh flows for enhanced security.
* **Custom User Logic:** Implemented a custom user model using email as the primary identifier combined with **Ethiopian phone validation**.
* **Clean API Design:** Exposed modular `/api/auth` endpoints for registration, login, and token refreshing to support frontend integration.

### Project Impact
* **Social Goal:** Designed to reunite individuals separated due to conflict or displacement.
* **Recognition:** Earned positive recognition from **GDG mentors** for technical execution and community relevance.
